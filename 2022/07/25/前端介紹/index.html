前端介紹

發表於2022-07-25更新於2022-07-25

引言

前後端分離是目前Web開發的主流模式，所謂的Web前端開發(Front-End Development)就是我們常指的網站或者網頁所展示的，包括内容，樣式以及交互部分。

網頁的組成

網站主要由3個部分組成，分別是HTML, CSS 和 JavaScript。

組件 功能
HTML: 負責網站的内容以及架構
CSS: 負責網站的樣式，我們稱之爲UI
JavaScript: 負責網站的動作，例如點擊跳轉，點擊按鈕功能等等

瀏覽器内核

瀏覽器内核是瀏覽器的核心，也是渲染引擎。

瀏覽器内核的作用是在於解析代碼、排版和渲染（render）頁面。不同的瀏覽器對網頁渲染不同，如果同一個網頁在不同的瀏覽器中打開，呈現的效果也有可能不同。

瀏覽器對應的内核

内核 瀏覽器 描述
Trident / EdgeHTML IE / Edge IE内核，是微软开发的一种排版引擎。
Gecko FireFox Gecko是当年最流行的排版引擎之一，仅次于Trident。
WebKit Safari WebKit特点在于源码结构清晰、渲染速度极快。缺点是对网页代码的兼容性不高
Chromium Chrome(2008) fork自Apple的Webkit内核，但把Webkit梳理得更有条理可读性更高
Blink Chrome(v28+) / Opera(v15+) Blink内核是WebKit内核中的WebCore组件的一个分支，即在WebCore的基础上进行二次开发

參考： 浏览器和内核

前端開發工具

開發工具是一個程序員重要的武器，挑選一個合適自己的開發工具真的很重要，極大的提升代碼效率，而我推薦的開發工具是 VS code。:thumbsup:

工欲善其事，必先利其器。

VS code 優點：

  1. 非常容易安裝插件
  2. 有一個龐大的社區支持幫助實現新功能
  3. 黑色主題很有高級感 （很重要）

可能也有些人喜歡功能齊全的IDE,例如 Jet 公司的 Intellij IDEA 和 Webstorm，或者更輕量化的Edictor 如 Vim(膜拜大佬)和Sublime。
這些可以自行比對，最重要的是自己用得順手。

VS code 快捷鍵：

快捷鍵 功能
Ctrl + Shift + N 開啟編輯器新視窗
Shift + Alt + F 代碼格式化
Alt + Up/Down 向 上/下 移動一行
Shift + Alt + Up/Down 快速複製向 上/下 一行代碼
Ctrl + S 儲存(很重要)
Ctrl + F 快速查找
Ctrl + H 快速替換
